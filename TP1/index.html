Decisiones de diseño

El equipo decidió utilizar una estética moderna y minimalista, combinando colores suaves con tonos azules para generar una interfaz clara, agradable y fácil de navegar.

También se eligió trabajar con tarjetas individuales para organizar mejor la información de cada integrante y mantener una estructura uniforme en todo el sitio web.

Se implementó un menú de navegación fijo para facilitar el acceso rápido a las distintas secciones del proyecto.

Dificultades encontradas

Una de las principales dificultades fue adaptar correctamente el diseño responsive para diferentes tamaños de pantalla.

También surgieron problemas al vincular las páginas individuales de cada integrante y organizar correctamente la estructura de carpetas del proyecto.

Estos problemas se resolvieron revisando las rutas de navegación, aplicando media queries en CSS y reorganizando los archivos siguiendo buenas prácticas de desarrollo web.

Cambios importantes

Durante el desarrollo se realizaron mejoras visuales en las tarjetas, botones y secciones principales del sitio.

Además, se agregaron efectos hover, sombras y transiciones para mejorar la experiencia del usuario y darle mayor dinamismo a la página.

Finalmente, se implementó JavaScript para mostrar mensajes dinámicos y personalizar el saludo principal según el horario del usuario.
